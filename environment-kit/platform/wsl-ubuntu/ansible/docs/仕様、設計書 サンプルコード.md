# サンプル Playbook / Role 抜粋（旧設計アーカイブ）

## site.yml（入口プレイブック）

```yaml
- hosts: localhost
  connection: local
  gather_facts: no
  tasks:
    - name: Install runtimes
      import_role:
        name: runtimes

    - name: Setup database
      import_role:
        name: database
```

---

## roles/runtimes/tasks/main.yml（開発ランタイムの例）

```yaml
# Node.js
- name: Install Node.js {{ lookup('config','nodejs_version') }}
  shell: |
    curl -fsSL https://deb.nodesource.com/setup_{{ lookup('config','nodejs_version').split('.')[0] }}.x | sudo -E bash -
    apt-get install -y nodejs={{ lookup('config','nodejs_version') }}*

# Go
- name: Install Go {{ lookup('config','go_version') }}
  unarchive:
    src: "https://go.dev/dl/go{{ lookup('config','go_version') }}.linux-amd64.tar.gz"
    dest: /usr/local
    remote_src: yes
  args:
    creates: "/usr/local/go/bin/go"
```

---

## roles/database/tasks/main.yml（PostgreSQL の例）

```yaml
- name: Install PostgreSQL {{ lookup('config','postgres_version') }}
  apt:
    name: "postgresql-{{ lookup('config','postgres_version') }}"
    state: present
    update_cache: yes

- name: Ensure dev DB exists
  postgresql_db:
    name: "{{ lookup('config','postgres_db_local') }}"
    encoding: "{{ lookup('config','postgres_db_encoding') }}"
    login_user: "{{ lookup('config','postgres_user') }}"
    login_password: "{{ lookup('config','postgres_password') }}"

- name: Ensure test DB exists
  postgresql_db:
    name: "{{ lookup('config','postgres_db_test') }}"
    encoding: "{{ lookup('config','postgres_db_encoding') }}"
    login_user: "{{ lookup('config','postgres_user') }}"
    login_password: "{{ lookup('config','postgres_password') }}"

- name: Enable extensions
  postgresql_ext:
    name: "{{ item }}"
    db: "{{ lookup('config','postgres_db_local') }}"
    login_user: "{{ lookup('config','postgres_user') }}"
    login_password: "{{ lookup('config','postgres_password') }}"
  loop: "{{ lookup('config','postgres_extensions').split(',') }}"
```

---

## ポイント

* **バージョンはすべて ansible.cfg から参照**
* **Node.js/Go などのランタイム**は tarball/apt 固定版をインストール
* **PostgreSQL** は DB 作成＋拡張機能インストールまで網羅
