# 03. 実装カタログ

本ドキュメントは、`pico-admin-template` の現行実装物を一覧化したもの。

---

## 1. 画面一覧

### 1.1 管理画面

| ファイル | 役割 |
|---|---|
| `src/index.html` | ダッシュボード。要約、KPI、短尺チャート、作業キュー |
| `src/pages/admin/analytics.html` | 流入分析。チャネル構成、セッション、成長キュー |
| `src/pages/admin/customers.html` | 顧客管理。アカウント一覧、CS キュー、拡張候補 |
| `src/pages/admin/news.html` | ニュース運用。記事カード、公開状態、例外対応 |
| `src/pages/admin/ajax.html` | 非同期サンプル。API 取得、検索、フィルタ、Create |
| `src/pages/admin/orders.html` | 受注管理。注文一覧、支払い状態、例外キュー |
| `src/pages/admin/todo.html` | Todo 管理。担当、期限、停滞タスク、作業キュー |
| `src/pages/admin/settings.html` | 設定。ワークスペース設定、権限、運用ポリシー |

### 1.2 認証 / 補助画面

| ファイル | 役割 |
|---|---|
| `src/pages/auth/signin.html` | サインイン |
| `src/pages/auth/signup.html` | アカウント作成 |
| `src/pages/auth/reset-password.html` | パスワード再設定 |
| `src/pages/errors/404.html` | Not Found |

## 2. CSS

| ファイル | 役割 |
|---|---|
| `src/css/pico.css` | Pico CSS 本体の CDN import |
| `src/css/admin-theme.css` | カラートークン、Pico 変数上書き、グローバル基調 |
| `src/css/admin-layout.css` | デスクトップファーストのレイアウト、ブレークポイント |
| `src/css/admin-components.css` | サイドバー、カード、テーブル、バッジ、フォームなど |

| クラス | 用途 |
|---|---|
| `.admin-shell` | 管理画面全体のグリッド |
| `.admin-sidebar` | 左サイドバー |
| `.admin-main` | 本文領域 |
| `.admin-header` | 画面ヘッダー |
| `.admin-content` | セクションラッパ |
| `.admin-card` | カード |
| `.admin-table` | データテーブル |
| `.admin-pager` | 一覧 pager |
| `.admin-badge--neutral` | 中立情報 |
| `.admin-badge--lime` | 正常 / 操作可能 |
| `.admin-badge--pink` | 警告 / 危険 |
| `.admin-chart--compact` | 圧縮チャート面 |
| `.admin-card--dense` | 省スペースカード |

## 6. src/package.json

現行の主要 script は以下。

| script | 用途 |
|---|---|
| `build` | CSS ビルド不要であることを明示 |
| `e2e` | Playwright 実行 |
| `e2e:headed` | headed 実行 |
| `e2e:ui` | UI モード |
| `e2e:codegen` | codegen 起動 |
