# Favicon Samples

用途別の `SVG favicon` サンプル。

- `admin-dashboard.svg`
  - 既定の管理画面用
- `news.svg`
  - ニュース / メディア用
- `ai-tools.svg`
  - AI ツール用
- `todo.svg`
  - Todo / タスク管理用
- `scheduler.svg`
  - スケジューラー / カレンダー用
- `sample-app.svg`
  - 汎用サンプルアプリ用

既定適用例:

```html
<link rel="icon" type="image/svg+xml" href="../../images/favicons/admin-dashboard.svg">
```
